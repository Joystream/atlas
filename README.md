## Getting Started

After cloning the repo, run:

```bash
$ cd atlas
$ yarn install
$ yarn start
```

To start the app on `localhost:3000`, Storybook on `localhost:6006` and the bundler in watch mode.

To build the app for distribution, run:

```bash
$ yarn build
```

To run tests (Currently WIP) run:

```bash
$ yarn test
```
